## Technical Architecture & Core Subsystems

### 1. High-Performance Multi-Artboard Canvas Engine
* **Figma-like UX:** Engineered a sophisticated, high-performance editor mimicking a Figma-style vector workflow built on top of **Fabric.js**[cite: 1].
* **Multi-Artboard Management:** Designed an advanced canvas state architecture enabling simultaneous handling, rendering, and interaction across multiple individual artboards within a single workspace context[cite: 1].
* **Asset Scalability:** Optimized real-time manipulation of text layers, high-resolution graphic arrays, shapes, and complex grouped visual elements[cite: 1].

### 2. Generative Multi-Agent Design Orchestration
* **AI-Driven Vector Design:** Integrated a robust multi-agent AI system utilizing **Gemini 3 Flash Preview** and **Claude APIs** to convert standard natural language instructions directly into structured, editable vector canvas designs for digital posters and advertisements[cite: 1].
* **Layout Optimization:** Implemented parallelized execution flows to process asset generation algorithms quickly and reliably directly within the Next.js backend ecosystem[cite: 1].

### 3. Automated Brand Guideline Extraction
* **URL Ingestion Engine:** Developed an intelligent brand scraping tool that evaluates a client's corporate identity directly from a public website URL[cite: 1].
* **Asset Extraction:** Implemented backend parsing pipelines that autonomously extract core brand identifiers—including exact brand color swatches, typographic structures, logo placement variables, and visual design styles—and loads them directly into the canvas editor configuration[cite: 1].

### 4. Enterprise Identity, Monetization & Token Ledger
* **Authentication:** Implemented secure user identity validation, session persistence, and authorization safeguards using **Better Auth**[cite: 1].
* **Multi-Tiered Subscription Engine:** Engineered a highly flexible billing architecture featuring granular subscription tiers (Usage-Based, Monthly, and Annual models) integrated via **Stripe (utilizing the modern 2026 Clover API)**[cite: 1].
* **Credit Ledger & Top-Up Flow:** Built an ad-hoc token purchase/top-up flow linked directly to user accounts, granting granular elastic credits to run heavy AI generation steps[cite: 1].
