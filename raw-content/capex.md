# Project Documentation: Capex Frontend

## Overview
- **Project Name:** Capex Dashboard
- **Role:** Frontend Lead Engineer[cite: 1]
- **Core Focus:** Legacy codebase migration, high-density state management, interactive financial calculations[cite: 1].
- **Tech Stack Evolution:** Migrated from a legacy JavaScript/Material-UI (MUI) foundation to a modern React, TypeScript, Vite, and shadcn/ui architecture[cite: 1].

---

## Technical Transformation & Legacy Migration
* **Architecture Modernization:** Spearheaded the comprehensive overhaul of a legacy JavaScript and older React codebase[cite: 1]. Migrated the entire application architecture to type-safe TypeScript (`.ts`/`.tsx`), optimizing build performance by switching from legacy bundling tools to Vite[cite: 1].
* **Design System Overhaul:** Replaced the heavy, older Material-UI (MUI) components with a utility-first, modern UI library built on Tailwind CSS and shadcn/ui[cite: 1].
* **Design Fidelity:** Successfully implemented high-fidelity user interfaces pixel-for-pixel from complex, raw Photoshop assets provided directly by the client[cite: 1].

---

## Core Engineering & Features

### 1. High-Density Multi-Step Calculation Engine
* **Complex Data Capture:** Engineered a multi-tiered data-entry framework requiring users to navigate heavily dependent input forms[cite: 1]. Supported interactive text fields, dynamic select matrices, and custom range sliders[cite: 1].
* **Dynamic Evaluation Logic:** Built an intricate mathematical client-side valuation engine that aggregates disparate variables, cross-references corporate metrics, and instantly computes comprehensive, tax-like financial output summaries based on complex regulatory algorithms[cite: 1].
* **State Orchestration:** Handled highly fluid component states using React Context API to ensure seamlessly cached progress across step boundaries without state fragmentation or unnecessary re-renders[cite: 1].

### 2. Multi-Level Authentication & Role-Based Access Control (RBAC)
* **Access Control:** Designed and deployed a rigid enterprise security structure by introducing designated Admin and User roles[cite: 1].
* **View Abstraction:** Restricted access to highly sensitive capital expenditure reports, configuration variables, and pricing calculations based on token-verified authorization scopes[cite: 1].

---

## Leadership & Engineering Impact
* **Technical Direction:** Acted as the Frontend Lead, delegating milestones to team members, managing features, and establishing strict code quality baselines through TypeScript strict mode[cite: 1].
* **Performance Gains:** Code modernization efforts achieved dramatic performance improvements, decreasing bundle sizes significantly and accelerating runtime interactive computation speeds across heavy multi-form arrays[cite: 1].
